Ce este Arduino?

Arduino este o platformă open-source utilizată pentru construirea de proiecte electronice. Arduino constă atât într-o placă de circuite programabile fizic (microcontroler), cât și într-o parte de software sau pe scurt IDE (Integrated Development Environment) care rulează pe computerul tău. Acesta este folosit pentru a scrie și a încărca codul computerului pe un board fizic.

Platforma Arduino este foarte populară în rândul oamenilor noi în domeniul electronicii. Spre deosebire de majoritatea plăcilor programabile anterioare, Arduino nu necesită hardware separat pentru a încărca cod nou pe placă. Un simplu cablu USB rezolvă această problemă. În cele din urmă, Arduino IDE folosește o versiune simplificată de C++, ceea ce face ușor să înveți cum să programezi. Cea mai folosită și documentată placă din întreaga familie Arduino este UNO R3.
